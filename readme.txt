== Description ==

This plugin is used for property and lot management system for realestate.You can add property details and  can add multiple lots details for one property.



== Installation ==


1. Upload `PLMS` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Look at your admin bar and enjoy using the new links there...
